FlipFeed effect tutorial

A few months ago, I did a FlipFeed demo. Now is the time to write a short tutorial to see how to create it on eXo Platform 3.5. Note that I write my post on the upcoming eXo Platform 3.5 (stay tuned for more info in the next weeks, it's coming) About the installation, follow the README.txt file, it's well described.

Add "news" Content Type

Follow my Steps :

- Log in using Mary (she has administrative rights in the Content section) - Go to Content Administration page -- in PLF 3.5, it's in the app menu on the right under Content section. - Go to Content Types tabs, then Manage Node Type
- To save time, I will upload a predefined nodetype definition Here is the link to the News nodetype definition : https://github.com/benjp/exosamples/blob/master/flipfeed/news-nodetype.xml It's a sample news content. The idea on this content type is to use an image that will be used in the FlipFeed effect.

Add the dialog template

Next we need a dialog to use this nodetype in the Content Explorer

Follow my Steps :

- In the Content Administation, switch to Content Presentation tab, then Manage Templates - Click on "Add" button in the bottom right. - Fill the fields to fit your needs (Label, Permissions, etc) - Add your Dialog template
The link for the Dialog template : https://github.com/benjp/exosamples/blob/master/flipfeed/dialog1.gtmpl

Add a default view

Finally, we need to add a default view to see this news in the website.

Follow my Steps :

- In the View tab, customize the View template (example below) - Save your template - Now, moment of truth, test it !
Link for the News view template : https://github.com/benjp/exosamples/blob/master/flipfeed/view1.gtmpl

Create a news

Just to test everything is okay :

- In the applications menu (in the admin bar), go to the Content Explorer - Select "Add Document", then "News", fill the fields - Save, you should see something like this

Import News sample set

Follow my Steps :

- Login as John and open Content Administration app - Go to > Content Presentation > Manage View - Edit "admin-view" - Click on Admin tab - Add "Import Node" action, then save - Open Content Explorer app - select "collaboration" drive, then click on Import Node action

Install the FlipFeed template

Follow my Steps :

- Open IDE app - In the menu, select > Window > Select workspace..., then dms-system workspace - Open in the left tree > dms-system > exo:ecm > views > templates > content-list-viewer > list - Create a new template (menu > File > Open Local File...)
The link for this template : https://github.com/benjp/exosamples/blob/master/flipfeed/FlipFeed.gtmpl
