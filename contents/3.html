How to add Youtube Content in 5 min ?

Sometimes, as we support mpg, mov, h264 videos in eXo Platform, this question pops up during demos : "Do you support Youtube videos ?" And I can only answer we don't have this "Content Type" in the default bundle. Ok, that's an easy answer, the real question is "Is it hard to support it ?". So, here you go, here is a 5 min tutorial to get Youtube videos in eXo Platform.

Download and Launch eXo

I encourage you to download Platform 3 to get the full User Experience : eXo Platform 3 or you can get eXo WCM Community Version on Source Forge : eXo WCM 2.1 Note that I write my post on the upcoming eXo Platform 3.5 (stay tuned for more info in the next weeks, it's coming) About the installation, follow the README.txt file, it's well described.

Add a new Content Type

Follow my Steps :

- Log in using Mary (she has administrative rights in the Content section) - Go to Content Administration page -- in PLF 3.5, it's in the app menu on the right under Content section. - Go to Content Types tabs, then Manage Node Type
- To save time, I will upload a predefined nodetype definition Here is the link to the Youtube Video nodetype definition : https://github.com/benjp/exosamples/blob/master/youtube/youtube-nodetype.xml It's a very simple one, the Youtube video key, the player width and height, that's all.

Add the dialog template

Next we need a dialog to use this nodetype in the Content Explorer

Follow my Steps :

- In the Content Administation, switch to Content Presentation tab, then Manage Templates - Click on "Add" button in the bottom right. - Fill the fields to fit your needs (Label, Permissions, etc) - Add your Dialog template
The link for the Dialog template : https://github.com/benjp/exosamples/blob/master/youtube/dialog1.gtmpl

Customize your Youtube view

Finally, we need to add the player to show the video in the website

Follow my Steps :

- In the View tab, customize the View template (example below) - Save your template - Now, moment of truth, test it !
Link for the Youtube view template : https://github.com/benjp/exosamples/blob/master/youtube/view1.gtmpl

You're done !

You can now add Youtube Videos in eXo Platform :

- In the applications menu (in the admin bar), go to the Content Explorer - Select "Add Document", then "Youtube Video" - You can enter either the youtube key directly or the url to the video. You can check in the view, I wrote it clever enough to work with both.
