FlipBoard effect in eXo WCM

Easy way to add a FlipBoard like effect in eXo WCM using Thumbnail Rest service.

In this video, I will show you how to :
  • Create cropped images
  • Add a new page template to create the FlipBoard effect

Create Cropped Images

That's actually easy to do in eXo WCM. Since WCM 2.2+ (the current trunk), we have a new REST service to create custom thumbnails. If you take this image in the Content Repository : http://localhost:8080/ecmdemo/rest-ecmdemo/jcr/repository/collaboration/Pictures/giraffe.jpg This "giraffe.jpg" image is stored in the Collaboration workspace under the Pictures folder. The upper link is the webdav access (using a REST service btw) to display this image. Side note : If it's a pdf file, this service will allow you to download it. Now if you want a small thumbnail of this image, call this service http://localhost:8080/ecmdemo/rest-ecmdemo/thumbnailImage/small/repository/collaboration/Pictures/giraffe.jpg The thumbnailImage REST service allows you to create different sizes :
  • small
  • medium
  • large
  • custom
In WCM 2.2+, the new one is "custom". By calling it, you can create all kind of cropped images of your original image and that's exactly the effect we want for the FlipBoard layout. So, just call : http://localhost:8080/ecmdemo/rest-ecmdemo/thumbnailImage/custom/640x0/repository/collaboration/Pictures/giraffe.jpg It will create a thumbnail with a 640px width (0 is used to ignore the height and keep the original proportion). Add a height of 100px, like this : http://localhost:8080/ecmdemo/rest-ecmdemo/thumbnailImage/custom/640x100/repository/collaboration/Pictures/giraffe.jpg You just created a banner that is a centered cropped image. It means it's not distorted and it's exactly what we want.
